# README Templates

This file contains complete README templates for different project types.

## Python CLI Application Template

```markdown
# {Project Name}

{One-sentence description of what the app does}

---

![Screenshot]({path-to-screenshot})

---

## Features

- **Feature 1**: Description
- **Feature 2**: Description
- **Feature 3**: Description

## Tech Stack

- **Python {version}+**
- **{Framework}**: {Purpose}
- **{Key Library}**: {Purpose}

## Getting Started

### Prerequisites

Ensure you have [{package manager}]({link}) installed.

### Installation

\`\`\`bash
# Install dependencies
{package-manager} install
\`\`\`

### Usage

Run the application:

\`\`\`bash
{command to run}
\`\`\`

Advanced commands:

\`\`\`bash
{command} --help
{example command 1}
{example command 2}
\`\`\`

## Development

### Running Tests

\`\`\`bash
{test command}
\`\`\`

### Code Quality

\`\`\`bash
# Linting
{lint command}

# Type Checking
{type check command}
\`\`\`

## Project Structure

\`\`\`text
{directory tree with comments}
\`\`\`

## License

{License info}

---
{Footer/Credits}
```

## Web Application Template

```markdown
# {Project Name}

{Tagline}

## Features

- Feature list

## Tech Stack

**Frontend:**
- {Framework/Library}

**Backend:**
- {Framework}
- {Database}

## Getting Started

### Environment Variables

Create a \`.env\` file:

\`\`\`env
DATABASE_URL=
API_KEY=
\`\`\`

### Installation

\`\`\`bash
npm install
\`\`\`

### Development

\`\`\`bash
npm run dev
\`\`\`

### Production Build

\`\`\`bash
npm run build
npm start
\`\`\`

## API Documentation

See [API.md](API.md) for endpoint documentation.

## Deployment

{Deployment instructions}
```
